# Course Compass

A simple app to track courses, assignments, and progress in one place.

## ⚡ Features

* Add and manage courses
* Track assignments and due dates
* Mark tasks as completed
* View progress for each course
* Change course colors to stay organized
* Check the AI generated time estimations to avoid falling behind
* Link with Canvas to bring over existing courses

## 🛠️ Tech Stack

* React (Frontend)
* Node + Express (Backend)
* MongoDB (Database)

## 📋 Usage

* Create a course or bring them over from Canvas
* Add assignments with due dates
* Mark them complete as you go
* Stay on top of your work
